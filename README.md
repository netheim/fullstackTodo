Simple TodoApp on NestJS, PrismaORM, GraphQL, PostgreSQL | React + TypeScript, Sass

1) Для запуска приложения необходимо установить все зависимости в обеих папках (client, server)
2) Также установить postgresql(pgAdmin) и в .env файле в папке server указать свои данные для подключения к БД.
3) Если все сделано правильно запускаем наш backend через - npm run start:dev, а затем frontend через npm run dev.
