Simple TodoApp on NestJS, PrismaORM, GraphQL, PostgreSQL | React + TypeScript, Sass
Для запуска приложения необходимо установить все зависимости в обеих папках (client, server), а также установить postgresql(pgAdmin) и в .env файле в папке server указать свои данные для подключения к БД.
Если все сделано правильно запускаеи наш backend через - npm run start:dev, а затем frontend через npm run dev.
